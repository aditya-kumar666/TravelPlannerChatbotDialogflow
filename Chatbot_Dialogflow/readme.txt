JPG file with name "Chatbot Assignment" is the flow chart of the “Travel Planner” chatbot, implement the same flow using Dialogflow.

1. Import TravelPlanner into dialogflow to work on or understand the chatbot.
2. "cloudfunction-code" folder contains the cloud function code which works on backend. You can use any cloud functions like Azure functions, Google's cloud function or Aws lambda. This example is written on Google's cloud function.
3. Chatbot recording video is the demonstration of different flows implemented in action. It is for better understanding.
